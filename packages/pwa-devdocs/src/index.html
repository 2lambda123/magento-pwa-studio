What is Magento PWA

A Progressive Web App, or PWA, is a website or web app that is built using performance-focused and mobile-centered implementation patterns. The Magento PWA project provides all the necessary support systems and tools to make a Magento storefront theme powered by PWA technologies.

See Magento PWA Overview for more information.
