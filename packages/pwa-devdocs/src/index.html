What is Magento PWA

"Progressive Web App", or PWA, is a term for a set of performance-focused and mobile-centered implementation patterns for Websites and Web apps. The Magento PWA project provides all the necessary support systems and tools to make a Magento 2 storefront theme powerd by PWA.

See Magento PWA Overview for more information.
